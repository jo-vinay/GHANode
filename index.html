Node js

Basic example that uses ExpressJS to serve a index.html file

Install ExpressJS
npm install g- express

index.js
Save the following as index.js

index.html
Create a html file named index.html and place in your desired content.

In command prompt
Start the local host by typing the following:
node sandbox/index.js

sandbox is the folder that I use to house my projects. It may be a different path on your computer.
you can try: node PathToYourProjectGoesHere/index.js
